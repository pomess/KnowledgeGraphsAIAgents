# CTTI

Centre de Telecomunicacions i Tecnologies de la Informació — the IT agency of the Catalan government, based in L'Hospitalet de Llobregat, Spain.

## Relevance

[[bruno-manzano|Bruno Manzano]] worked as a Machine Learning Engineer at CTTI from October 2024 to September 2025. His work focused on:

- **Code defect prediction** — AI models to predict defects in software.
- **Vulnerability detection** — identifying security vulnerabilities using ML.
- **Automated test case generation** — generating test cases automatically.
- **Production monitoring** — tools for automatic problem detection in live environments.
- **Predictive maintenance** — models to anticipate system failures before they occur.

This role represented Bruno's deepest work in ML engineering and software quality automation before moving to the more agent-focused work at [[deloitte|Deloitte]].
